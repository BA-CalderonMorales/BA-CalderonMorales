# Projects

## Original Work

Projects I created from scratch.

### Terminal Jarvis

CLI orchestration for multiple AI assistants and team workflows, designed for mobile-first development environments.

**Why:** Serious work should not require a desk. I built this because I wanted to use Claude, Gemini, and other AI tools from my phone without losing session context.

**Key features:**
- Session continuation across authentication flows
- Multi-tool orchestration
- Mobile-optimized TUI

[github.com/BA-CalderonMorales/terminal-jarvis](https://github.com/BA-CalderonMorales/terminal-jarvis)

### Agent Harness

A clean-room, pattern-derived agent harness for building coding agents.

**Why:** I wanted to understand agent architectures from first principles. This is my research project turned tool.

**Key features:**
- Tool dispatch with permission controls
- Two execution modes: interactive and yolo
- Secure credential storage

[github.com/BA-CalderonMorales/agent-harness](https://github.com/BA-CalderonMorales/agent-harness)

### Lumina

An isolated runtime for agentic workflows.

**Why:** Built for operators who want the power of AI automation without renting their toolchain. No subscriptions. No lock-in. Just precise, non-human execution that respects boundaries and keeps your data yours.

**Status:** Private repository. In active development.
